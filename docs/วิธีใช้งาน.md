##  การติดตั้งและรันระบบ (Installation & Setup)
### 1. การติดตั้ง Library ที่จำเป็น
    ก่อนเริ่มใช้งานครั้งแรก ให้ติดตั้งแพ็กเกจทั้งหมดผ่านไฟล์ `requirements.txt`: pip install -r requirements.txt
### 2. การรันระบบในเครื่องคอมพิวเตอร์
    uvicorn backend.main:app --reload
### 3. การรันระบบบน Cloud (Production - Render)
    gunicorn -w 4 -k uvicorn.workers.UvicornWorker backend.main:app

# ขอบเขตการทำงานของระบบ                                                                                                                                                                 ** ส่วนของผู้ใช้งานทั่วไป       (User)                                                                                                                                                                                 
*   สมัครสมาชิก/เข้าสู่ระบบ: รองรับการลงทะเบียนและยืนยันตัวตนด้วย Token
*   จัดการงาน: เพิ่ม, แก้ไข, ลบ และอัปเดตสถานะงาน (To-do, In-progress, Done)

*   ระบบแจ้งเตือน: แจ้งเตือนงานที่ใกล้ถึงกำหนดส่งอัตโนมัติ
*   โปรไฟล์: แก้ไขข้อมูลส่วนตัวและรูปโปรไฟล์ได้
** ส่วนของผู้ดูแลระบบ(Admin)
* หน้าสถิติ (Stats): ดูภาพรวมจำนวนผู้ใช้, จำนวนงานทั้งหมด และอัตราความสำเร็จของคนทั้งระบบ  
* การจัดการผู้ใช้: ตรวจสอบรายชื่อผู้ใช้งานทั้งหมดและสามารถลบผู้ใช้งานออกจากระบบได้
